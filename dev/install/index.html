Installation Instructions

Kinetic is a registered Julia package in the official entry. We recommend installing it with the built-in Julia package manager. It automatically installs a currently stable and tagged release. From the Julia REPL, you can add the package, instantiate and build all the dependencies via

julia> ]
(v1.5) pkg> add Kinetic
(v1.5) pkg> instantiate
(v1.5) pkg> build Kinetic

This will install Kinetic and all its dependencies. After that, we can using or import the package,

julia> using Kinetic

Kinetic can be updated to the latest tagged release from the package manager by executing

(v1.5) pkg> update Kinetic
Use Julia 1.3 or newer

Kinetic matches perfectly with Julia 1.3 and newer versions. Installing it with an older version of Julia will locate incomplete functionality.
